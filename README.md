# KRS DB
### Бекенд микросервис, который взаимодействует с остальными сервисами и фронтендом
### Именно через него осуществляется деплой

# Описание системы

**KRS** - учебный проект, созданный для исследования микросервисной архитектуры и аутентификации через JWT
